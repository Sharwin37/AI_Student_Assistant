# AI-Student-Assistant

AI Powered Smart Campus Assistant Project


This project is based on Artificial Intelligence and Machine Learning concepts.  
It helps students to predict their marks, generate study plans, and get guidance using a chatbot.


You can see my report.pdf for more details.


## Features
- Predict student marks
- Check pass/fail result
- Generate study plan
- View student clusters
- Chatbot for guidance


## How to run

1. Install Python

2. Install required libraries:
pip install pandas scikit-learn matplotlib

3. Run the project:
python src/main.py


## Project Structure

src → main code files  
data → dataset  
prolog → rules
